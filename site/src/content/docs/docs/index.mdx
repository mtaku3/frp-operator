---
title: frp-operator docs
description: Kubernetes operator that gives every Service a public IP via frps.
---

frp-operator provisions cheap cloud frps exits and bin-packs your tunnels onto them, so any Kubernetes Service can have a routable address regardless of inbound network constraints.

These docs follow [Diátaxis](https://diataxis.fr):

- **Getting started** — step-by-step tutorials for first install and first tunnel.
- **Guides** — recipe-style how-to pages, one task per page.
- **Concepts** — explanations of the model (CRDs, scheduling, disruption).
- **Reference** — auto-generated CRD fields, Helm values, and operator flags.

If you're new, start with [Install on kind](/docs/getting-started/install-kind).
